# Whatsapp Chat Analyzer

This is a web app that allows you to analyze your Whatsapp chat data in the form of graphs and tables.

You have to first export the chat data of a group or individual from Whatsapp and then upload the data in the app. We display the analysis in the such a form that you can easily understand the data.

# How to use the app

### 1. Create a vertual environment and activate it

```bash
python -m venv venv
```

For Windows:

```bash
venv\Scripts\activate
```

For Mac and Linux:

```bash
source venv/bin/activate
```

### 2. Install the required packages

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
