# Olympics Data Analysis

This is a web app that allows you to analyze the data of all the olympic games from 1896 to 2016.

The data is represented in the form of graphs and tables such that you can easily understand the data.

# How to use the app

### 1. Download the datasets

Create a folder named `data` in the root directory and add the two datasets that you'll get from the link below

Dataset Link: https://www.kaggle.com/heesoo37/120-years-of-olympic-history-athletes-and-results

### 2. Create a vertual environment and activate it

```bash
python -m venv venv
```

For Windows:

```bash
venv\Scripts\activate
```

For Mac and Linux:

```bash
source venv/bin/activate
```

### 2. Install the required packages

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
